📁 프로젝트 파일 정리 완료 (2025-01-03)

📦 기타/ - 프로젝트 관련 자료 폴더
  │
  ├─ 📚 docs/ - 문서 폴더
  │   ├─ 프로젝트_파일_구조_가이드.md  ← ★ 전체 프로젝트 구조 상세 가이드
  │   ├─ README_SIMPLE.txt
  │   ├─ DB연동_작업_매뉴얼.md
  │   ├─ DB연동_빠른시작.txt
  │   └─ API_쉬운_설명.txt
  │
  ├─ 🚀 scripts/ - 실행 스크립트 폴더
  │   ├─ 0_INSTALL.bat           ← 최초 설치
  │   ├─ START_ALL.bat           ← 개발 모드 실행
  │   ├─ BUILD_AND_RUN.bat       ← 빌드 후 실행
  │   └─ RUN_PRODUCTION.bat      ← 프로덕션 실행
  │
  └─ 🗄️ database/ - 데이터베이스 스키마 폴더
      ├─ create_tables.sql       ← PostgreSQL용
      └─ create_tables_mysql.sql ← MySQL용

═══════════════════════════════════════════════════════════

🎯 빠른 시작 가이드

1. 처음 사용하는 경우:
   → 기타/scripts/0_INSTALL.bat 실행
   → 기타/scripts/START_ALL.bat 실행

2. 프로젝트 구조를 알고 싶은 경우:
   → 기타/docs/프로젝트_파일_구조_가이드.md 읽기

3. 데이터베이스 연동하려는 경우:
   → 기타/database/create_tables.sql 실행
   → 기타/docs/DB연동_작업_매뉴얼.md 참고

4. API 테스트하려는 경우:
   → 기타/docs/API_쉬운_설명.txt 참고
