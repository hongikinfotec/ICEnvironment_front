# 🚀 실행 스크립트 폴더

이 폴더에는 프로젝트 설치 및 실행을 위한 배치 파일들이 있습니다.

## 📄 파일 목록

| 파일명 | 용도 | 실행 순서 |
|--------|------|----------|
| `0_INSTALL.bat` | 전체 설치 | **1단계** |
| `START_ALL.bat` | 개발 모드 실행 | **2단계** |
| `BUILD_AND_RUN.bat` | 빌드 후 실행 | 프로덕션 배포 시 |
| `RUN_PRODUCTION.bat` | 프로덕션 실행 | 빌드 후 실행 시 |

## 🎯 사용 방법

### 처음 설치할 때 (최초 1회)
1. **0_INSTALL.bat** 더블클릭
   - Node.js 패키지 설치
   - Python 패키지 설치
   - 완료까지 약 5-10분 소요

### 개발하면서 테스트할 때 (매일)
2. **START_ALL.bat** 더블클릭
   - 프론트엔드 개발서버 시작 (Port 5173)
   - 백엔드 API 서버 시작 (Port 8000)
   - 브라우저 자동 열림

### 실제 배포할 때
3. **BUILD_AND_RUN.bat** 더블클릭
   - 프론트엔드 빌드 (최적화)
   - 프로덕션 서버 시작
   - 빌드 완료까지 약 1-2분

또는

3. **RUN_PRODUCTION.bat** 더블클릭
   - 이미 빌드된 파일로 서버 시작
   - 즉시 실행

## ⚠️ 주의사항

- Windows 전용 배치 파일입니다
- 실행 전에 Node.js와 Python이 설치되어 있어야 합니다
- 관리자 권한 불필요 (일반 사용자로 실행 가능)

## 🔧 문제 해결

### "Node.js를 찾을 수 없습니다" 에러
→ Node.js 18 이상 설치 필요
→ https://nodejs.org/

### "Python을 찾을 수 없습니다" 에러
→ Python 3.9 이상 설치 필요
→ https://python.org/

### 포트가 이미 사용 중입니다
→ 다른 프로그램이 5173 또는 8000 포트 사용 중
→ 해당 프로그램 종료 후 재시도
