========================================
Wastewater Monitoring System
SIMPLE GUIDE
========================================

2가지 실행 방법이 있습니다:

========================================
방법 1: 개발 모드 (2개 서버)
========================================
- 백엔드/프론트엔드 따로 실행
- 코드 수정하면서 개발할 때 사용

[처음 1번만]
  0_INSTALL.bat

[실행]
  START_ALL.bat  (자동으로 2개 서버 실행)

또는

  1_START_BACKEND.bat
  2_START_FRONTEND.bat


========================================
방법 2: 배포 모드 (1개 서버만!) ★추천★
========================================
- 실제 사용할 때 이 방법 사용
- 파일 1개만 실행하면 끝!

[처음 1번만]
  0_INSTALL.bat
  BUILD_AND_RUN.bat  (프론트엔드 빌드)

[실행]
  RUN_PRODUCTION.bat  (이것만 클릭!)

[접속]
  http://localhost:8000


========================================
비교
========================================

개발 모드:
  - 2개 창 실행 필요
  - 포트: 8000 (백엔드), 5173 (프론트)
  - 코드 수정 시 자동 반영
  - 개발자용

배포 모드: ★
  - 1개 창만 실행
  - 포트: 8000 (통합)
  - 빠르고 안정적
  - 실사용자용


========================================
추천 방법
========================================

실제 사용: RUN_PRODUCTION.bat
개발/수정: START_ALL.bat
